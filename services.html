服務介紹

中國代購：

淘寶 / 閒魚 / 天貓 / 京東 / 拚多多 / 1688 / 微店 / B站會員購 等

日本代購：

アニメイト / ゲーマーズ / とらのあな / BOOTH / 樂天 / Amazonjp / COSPA 等

沒有提到的網站都可以問問看，日本那邊有一些不送轉運（像是 amiami，ami 有特別設國際版網站我也能幫忙買，不過稅費自負）

收費：

(商品價格 + 國內運費) 匯率 * 1.1 + 國際運費，有需要幫忙溝通的話 * 1.15

國際運費：

  • 中國：50 元 / 1 公斤（包稅）
  • 日本：220 元 / 1 公斤（包稅）

服務流程：

  1. 社團開單
  2. 填單詢問 → 報價
  3. 後續統一 → 確認匯款 → 商品購買 → 寄送回台 → 賣貨便二補國際運費寄出

（特殊體積需要走郵局的話運費另計）
