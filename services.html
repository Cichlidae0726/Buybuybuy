服務介紹

中國代購： 淘寶 / 閒魚 / 天貓 / 京東 / 拚多多 / 1688 / 微店 / B站會員購 等

日本代購： アニメイト / ゲーマーズ / とらのあな / BOOTH / 樂天 / Amazonjp / COSPA 等

沒有提到的網站都可以問問看，日本有些不送轉運（如 amiami）也能協助處理，稅費自負。

收費： (商品價格 + 國內運費) 匯率 x 1.1，有需要協助溝通則 x 1.15

國際運費：

  • 中國：50元 / 1公斤（包稅）
  • 日本：220元 / 1公斤（包稅）

流程：

  1. 社團開單
  2. 填單詢問 → 報價
  3. 統一處理 → 匯款 → 購買 → 回台 → 寄出

（特殊體積使用郵局寄送，運費另計）
